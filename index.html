Event Loop

First JavaScript code runs in call stack. Call stack read code line by line. When it's get a asynchronous code, it's immediately pass the code in assist api and it's go next code. When asynchronous code is ready then api push the code in callback queue. When call stack is done his work then event loop push the code in call stack and code execute.

JavaScript Work

JavaScript is a single-threaded language, which means it has only one call stack that is used to execute the program. The source code is passed through a program called a compiler, which translates it into byte code that the machine understands and can execute. JavaScript is a client-side scripting language and one of the most efficient, commonly used scripting languages.
